
API
===

Esse tópico é reservado para guardar informações sobre cada versão da API do
SolarEDX. A utilização de versionamento é importante para permitir maior 
flexibilidade durante o desenvolvimento e a manutenção do software. 

É recomendada a leitura do artigo :ref:`quickstart` antes da leitura da 
documentação da API.

Atualmente existe apenas a versão ``dev`` (desenvolvimento).

A API do SolarEDX utiliza a biblioteca 
`Tastypie <django-tastypie.readthedocs.org>`_.
